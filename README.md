# рҹҺө Just Solo

**и·Ёе№іеҸ°йҹід№җж’ӯж”ҫеҷЁ** вҖ”вҖ” еҹәдәҺ Qt 6.8.3 дёҺ QML жһ„е»әпјҢиҝҪжұӮжһҒиҮҙжҖ§иғҪдёҺдјҳйӣ…дҪ“йӘҢгҖӮ

---

## рҹ“– йЎ№зӣ®з®Җд»Ӣ

Just Solo жҳҜдёҖж¬ҫ**иҪ»йҮҸгҖҒй«ҳжҖ§иғҪгҖҒз•ҢйқўзІҫзҫҺ**зҡ„жЎҢйқўйҹід№җж’ӯж”ҫеҷЁгҖӮе®ғдҪҝз”Ё **C++ + QML** д»Һйӣ¶жһ„е»әпјҢдёҚдҫқиө–д»»дҪ• Web иҝҗиЎҢж—¶пјҢеңЁдҝқжҢҒзҺ°д»ЈеҢ– UI зҡ„еҗҢж—¶пјҢе®һзҺ°дәҶжһҒдҪҺзҡ„еҶ…еӯҳеҚ з”Ёе’ҢжөҒз•…зҡ„дәӨдә’е“Қеә”гҖӮ

### дёәд»Җд№ҲйҖүжӢ© Qt Quickпјҹ

| з»ҙеәҰ | е…ёеһӢ Web жҠҖжңҜж Ҳ (Electron) | Qt Quick (Just Solo) |
|------|---------------------------|------------------------|
| еҶ…еӯҳеҚ з”Ё | 300MB+ | < 80MB |
| еҗҜеҠЁж—¶й—ҙ | ~3s | < 0.5s |
| еҢ…дҪ“з§Ҝ | 150MB+ | ~40MB |
| UI жёІжҹ“ | жөҸи§ҲеҷЁеҗҲжҲҗ | GPU еҺҹз”ҹеҠ йҖҹ |
| йҹійў‘з®Ўзәҝ | Web Audio API (еҸ—йҷҗ) | FFmpeg + PortAudio (е…Ёж јејҸ) |
| зі»з»ҹйӣҶжҲҗ | жңүйҷҗ | еҺҹз”ҹзі»з»ҹжүҳзӣҳгҖҒеӘ’дҪ“й”®гҖҒйҖҡзҹҘ |

жң¬йЎ№зӣ®ж—ЁеңЁеұ•зӨә **Qt Quick** еңЁж¶Ҳиҙ№зә§жЎҢйқўеә”з”Ёдёӯзҡ„ејәеӨ§иғҪеҠӣпјҢе№¶жҸҗдҫӣдёҖдёӘ**й«ҳеҸҜжү©еұ•гҖҒжЁЎеқ—жё…жҷ°**зҡ„д»Јз Ғжһ¶жһ„пјҢж–№дҫҝејҖеҸ‘иҖ…дәҢж¬ЎејҖеҸ‘гҖӮ

#### еҗ„еұӮиҒҢиҙЈ

**UI еұӮ (QML + JavaScript)**
- зәҜеЈ°жҳҺејҸзј–еҶҷпјҢжүҖжңүз•Ңйқўе…ғзҙ йҖҡиҝҮ `Item` / `Rectangle` / `Text` зӯүеҹәзЎҖз»„д»¶з»„еҗҲиҖҢжҲҗгҖӮ
- еҠЁз”»зі»з»ҹеҹәдәҺ `NumberAnimation`гҖҒ`Behavior`гҖҒ`Transition` е®һзҺ°пјҢе…ЁйғЁз”ұ GPU еҠ йҖҹгҖӮ
- дё»йўҳйўңиүІйҖҡиҝҮ QML еҚ•дҫӢ `ThemeManager` жіЁе…ҘпјҢеҲҮжҚўдё»йўҳж—¶жүҖжңүз»‘е®ҡиҮӘеҠЁжӣҙж–°пјҢж— йңҖжүӢеҠЁеҲ·ж–°гҖӮ
- еӣҪйҷ…еҢ–еӯ—з¬ҰдёІйҖҡиҝҮ `qsTr()` е®Ҹж Үи®°пјҢй…ҚеҗҲ Qt Linguist е·Ҙе…·й“ҫеҜјеҮә `.ts` зҝ»иҜ‘ж–Үд»¶гҖӮ

**дёҡеҠЎйҖ»иҫ‘еұӮ (C++ ж ёеҝғеҜ№иұЎ)**
- жүҖжңүж ёеҝғеҜ№иұЎйҖҡиҝҮ `qmlRegisterSingletonInstance()` жҲ– `qmlRegisterType()` жҡҙйңІз»ҷ QMLгҖӮ
- дёҡеҠЎеҜ№иұЎй—ҙйҖҡиҝҮ `EventBus`пјҲеҹәдәҺ `QObject::event()` жҲ–дҝЎеҸ·/ж§Ҫе№ҝж’ӯпјүе®һзҺ°жқҫиҖҰеҗҲйҖҡдҝЎгҖӮ
- жҜҸдёӘж ёеҝғеҜ№иұЎдёә `QObject` еӯҗзұ»пјҢдҪҝз”Ё `Q_PROPERTY` е®ҸжҡҙйңІеұһжҖ§пјҢQML дҫ§еҸҜзӣҙжҺҘз»‘е®ҡгҖӮ

**еҹәзЎҖжңҚеҠЎеұӮ (C++ жңҚеҠЎз»„д»¶)**
- жүҖжңүеӨ–йғЁдҫқиө–пјҲи§Јз ҒгҖҒиҫ“еҮәгҖҒж ҮзӯҫгҖҒзҪ‘з»ңпјүеқҮйҖҡиҝҮжҺҘеҸЈжҠҪиұЎпјҢдёҡеҠЎеұӮд»…дҫқиө–жҺҘеҸЈеӨҙж–Үд»¶гҖӮ
- е…·дҪ“е®һзҺ°еңЁиҝҗиЎҢж—¶йҖҡиҝҮдҫқиө–жіЁе…Ҙжһ„йҖ еҮҪж•°жҲ–е·ҘеҺӮж–№жі•дј е…ҘпјҢеҸҜйҡҸж—¶жӣҝжҚўгҖӮ
- жҜҸдёӘжңҚеҠЎиҝҗиЎҢеңЁзӢ¬з«ӢзәҝзЁӢпјҲеҰӮи§Јз ҒзәҝзЁӢгҖҒзҪ‘з»ңиҜ·жұӮзәҝзЁӢпјүпјҢйҒҝе…Қйҳ»еЎһ UIгҖӮ

### 4. EventBus дәӢд»¶жҖ»зәҝ

`EventBus` жҳҜе…ЁеұҖеҚ•дҫӢпјҢжҸҗдҫӣеҸ‘еёғ/и®ўйҳ…жЁЎејҸзҡ„и·ЁжЁЎеқ—йҖҡдҝЎгҖӮжүҖжңүжЁЎеқ—ж— йңҖдә’зӣёжҢҒжңүеј•з”ЁпјҢеҸӘдҫқиө–дәӢд»¶зұ»еһӢеҚіеҸҜи§ЈиҖҰеҚҸдҪңгҖӮ

**ж ёеҝғжҺҘеҸЈпјҡ**
```cpp
class EventBus : public QObject {
public:
    static EventBus* instance();

    // еҸ‘еёғдәӢд»¶пјҲзәҝзЁӢе®үе…Ёпјү
    void publish(const QString& eventType, const QVariantMap& payload);

    // и®ўйҳ…дәӢд»¶пјҢиҝ”еӣһи®ўйҳ… ID з”ЁдәҺеҸ–ж¶Ҳ
    int subscribe(const QString& eventType,
                  std::function<void(const QVariantMap&)> handler);

    // еҸ–ж¶Ҳи®ўйҳ…
    void unsubscribe(int subscriptionId);
};
```

**дё»иҰҒдәӢд»¶зұ»еһӢпјҡ**

| дәӢд»¶зұ»еһӢ | еҸ‘еёғиҖ… | и®ўйҳ…иҖ… | иҪҪиҚ· |
|---------|--------|--------|------|
| `player.state.changed` | PlayerController | LyricsView, PlaylistView, зі»з»ҹжүҳзӣҳ | `{state, oldState}` |
| `player.position.changed` | PlayerController | ProgressSlider, LyricsView | `{position, duration}` |
| `player.track.changed` | PlayerController | LyricsView, PlaylistView, е°ҒйқўжҳҫзӨә | `{metadata}` |

| `theme.changed` | ThemeManager | е…ЁеұҖ | `{themeName}` |
| `settings.changed` | SettingsManager | зӣёе…іи®ўйҳ…иҖ… | `{key, value}` |

**и®ҫи®ЎдјҳеҠҝпјҡ**
- ж–°еўһеҠҹиғҪеҸӘйңҖи®ўйҳ…е·ІжңүдәӢд»¶пјҢж— йңҖдҝ®ж”№еҸ‘еёғз«Ҝд»Јз ҒгҖӮ

#### 6.2 LibraryScanner

## вңЁ ж ёеҝғзү№жҖ§

- **й«ҳжҖ§иғҪйҹійў‘еј•ж“Һ** вҖ” еҹәдәҺ FFmpeg + PortAudio зҡ„жЁЎеқ—еҢ–и§Јз Ғ/иҫ“еҮәз®ЎзәҝпјҢж”ҜжҢҒ MP3/FLAC/AAC/OGG/WAV/WMA/OPUS/APE зӯүж јејҸ
- **жөҒз•…зҡ„ QML з•Ңйқў** вҖ” GPU еҠ йҖҹжёІжҹ“пјҢ60fps еҠЁз”»пјҢжүҖжңүиҝҮжёЎж•ҲжһңзЎ¬д»¶еҠ йҖҹ
- **дё»йўҳзі»з»ҹ** вҖ” еҶ…зҪ®жҡ—иүІ/дә®иүІдё»йўҳпјҢеҹәдәҺ QML еҚ•дҫӢзҡ„еұһжҖ§з»‘е®ҡе®һзҺ°йӣ¶е»¶иҝҹеҲҮжҚў
- **жӯҢиҜҚжҳҫзӨә** вҖ” ж”ҜжҢҒ LRC жң¬ең°и§ЈжһҗгҖҒеҶ…еөҢжӯҢиҜҚгҖҒеңЁзәҝжҗңзҙўпјҢеҚЎжӢүOKйЈҺж јйҖҗиЎҢ/йҖҗеӯ—й«ҳдә®
- **еӘ’дҪ“еә“з®ЎзҗҶ** вҖ” еҗҺеҸ°иҮӘеҠЁжү«жҸҸпјҢSQLite зј“еӯҳзҙўеј•пјҢе®һж—¶зӣ‘еҗ¬ж–Үд»¶еҸҳеҢ–
- **10 ж®өеқҮиЎЎеҷЁ** вҖ” еҹәдәҺ Biquad IIR ж»ӨжіўеҷЁпјҢеҶ…зҪ®еӨҡз§Қйў„и®ҫпјҢж”ҜжҢҒиҮӘе®ҡд№үеӯҳиҜ»
- **жҸ’д»¶жһ¶жһ„** вҖ” йў„з•ҷжү©еұ•жҺҘеҸЈпјҢж”ҜжҢҒеҠЁжҖҒеҠ иҪҪ `.dll/.so/.dylib` жҸ’д»¶
- **дәӢд»¶жҖ»зәҝ** вҖ” еҸ‘еёғ/и®ўйҳ…жЁЎејҸи§ЈиҖҰжЁЎеқ—йҖҡдҝЎпјҢжҳ“дәҺжү©еұ•ж–°еҠҹиғҪ
- **ж’ӯж”ҫеҲ—иЎЁ** вҖ” ж”ҜжҢҒжӢ–жӢҪжҺ’еәҸгҖҒm3u/pls еҜје…ҘеҜјеҮәгҖҒж–ӯзӮ№з»ӯж’ӯ
- **зі»з»ҹйӣҶжҲҗ** вҖ” зі»з»ҹжүҳзӣҳгҖҒеӘ’дҪ“й”®пјҲй”®зӣҳеҝ«жҚ·й”®пјүгҖҒеҺҹз”ҹйҖҡзҹҘ
- **и·Ёе№іеҸ°** вҖ” ж”ҜжҢҒ Windows 10+гҖҒmacOS 12+гҖҒLinux (X11/Wayland)
